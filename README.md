# Employee Management System - Backend

## 📌 Overview
This is a Spring Boot backend application for managing employees.

## 🚀 Tech Stack
- Java 21
- Spring Boot 3
- Spring Data JPA
- MySQL
- REST APIs

## ✅ Features
- Create Employee
- Read All Employees
- Update Employee
- Delete Employee
- Search Employee by Email

## 🛠 How to Run

1. Clone the repository
2. Configure database in application.properties
3. Run EmsApplication.java
4. Access APIs at:
   http://localhost:8080/api/employees

## 📂 Project Structure
- controller
- service
- repository
- entity
